# Simple_Calculator
A basic calculator web app built using HTML, CSS, and JavaScript. It can perform simple arithmetic operations like addition, subtraction, multiplication, and division. Includes features like clear (AC), delete (DEL), and a clean user interface.

Features

1. Addition, Subtraction, Multiplication, Division
2. Clear button (AC) to reset input
3. Delete button (DEL) to remove last digit
4. Supports decimal numbers
5. Simple and responsive design

Technologies Used

1. HTML
2. CSS
3. JavaScript
